# 三问工作划分

## 共享计算核心

`src/heliostat/` 负责三问共同使用的物理模型：

- 太阳位置和 DNI；
- 镜面法向、局部坐标系和余弦效率；
- 大气透射率；
- 阴影遮挡射线追踪；
- 太阳锥光截断追踪。

后续第 2、3 问应调用这套核心，不另写一份光学公式。

## 第一问：给定镜场评价

状态：已实现。

入口：`src/solve_q1.py`（兼容入口，实际实现位于 `src/heliostat/q1/`）

工作内容：

1. 读取 `task/A/fj.xlsx` 的 1745 个坐标；
2. 按给定的 `6.2 m × 6.2 m`、安装高度 `4.5 m` 和中心塔计算 60 个时刻；
3. 输出逐时刻、月平均、年平均和单镜年平均结果；
4. 完成反射定律、效率范围、单镜遮挡、集热器尺寸和采样收敛检查。

第一问只做评价，不改变塔位置、镜面尺寸、数量和坐标。
逐时刻结果只保留全场汇总；单镜指标在计算过程中流式累加，最终只保存 1745 行年平均结果。

第一问论文材料固定为：

- 月平均综合光学效率与单位面积输出热功率组合图；
- 单镜年平均综合光学效率空间分布图；
- 月平均结果表；
- 年平均结果表；
- 一张数值验证表。

不单独生成纯镜场布局图或统一光学模型示意图，也不保存单镜逐时刻明细。
正式建模、结果和与第二问关系集中写在 `docs/questions/第一问.md`；
完整公式集中写在 `docs/questions/第一问公式说明.md`；其余三份 `q1-*`
文档作为实施、详细推导和验证附件。

## 第二问：统一规格镜场优化

状态：已实现并形成正式扁平交付包。

建议入口：`src/solve_q2.py`

设计变量：

- 吸收塔位置 `(x_T, y_T)`；
- 统一镜面宽度、高度和安装高度；
- 定日镜数量；
- 各定日镜位置。

约束：

- 年平均输出热功率不低于 `42 MW`；
- 镜面边长、安装高度、场地边界、塔周禁布区和最小中心距满足题面；
- 所有镜面使用统一尺寸和安装高度。

目标：

```text
最大化 单位镜面面积年平均输出热功率
```

当前实现：

1. 并行生成分区交错同心圆和三分区改进 Campo 镜场；
2. 在光学评价前检查尺寸、触地、场地、禁区和严格中心距，并统一保留 0.01 m 安全余量；
3. 两种布局分别进行 Sobol 分散初值和多起点循环变步长搜索；
4. 通过粗步定位、区间回扫和 `K±2` 确定镜场外边界；
5. 使用问题一正式精度复算两种布局并比较；
6. 对胜出布局执行外层东西对称镜位修剪；
7. 写出坐标、月年平均结果、论文表、四张正式图片和提交用 Excel；
8. 生成无需工程包即可独立运行的 `outputs/q2/01_第二问完整代码.py`。

正式结果只能来自完整 60 时刻与正式数值精度运行；`--smoke` 输出仅用于验证程序链路。
当前正式 Campo 方案含 1469 面镜子，年平均输出为 `42.044238 MW`；加密复算为 `42.055115 MW`。
正文版优化方案和结果集中写在 `docs/questions/第二问.md`，其中分区同心圆
只作强候选简述，最终 Campo 方案详细展开；完整公式总表写在
`docs/questions/第二问公式说明.md`，推导背景与搜索细节保留在
`docs/questions/q2-technical-notes.md`。

## 第三问：可变规格镜场优化

状态：待实现。

建议入口：`src/solve_q3.py`

在第二问变量基础上，每面定日镜允许拥有不同的：

- 镜面宽度和高度；
- 安装高度。

直接优化数千个连续变量不可控，建议先按半径、方位或效率分区，让同一区域共享一组尺寸和高度；得到稳定方案后，再对边界镜和低效率镜做局部调整。

实现顺序：

1. 复用第二问的塔位置和布局生成器；
2. 增加分区规格参数；
3. 以总面积最小/单位面积功率最大为目标做分层优化；
4. 剔除低贡献镜并重新满足 `42 MW`；
5. 用正式精度复算并写出 `result3.xlsx`。
