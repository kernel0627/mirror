# 三问工作划分

## 共享计算核心

`src/heliostat/` 负责三问共同使用的物理模型：

- 太阳位置和 DNI；
- 镜面法向、局部坐标系和余弦效率；
- 大气透射率；
- 阴影遮挡射线追踪；
- 太阳锥光截断追踪；
- 单镜效率、镜场功率和月均/年均汇总。

后续第 2、3 问应调用这套核心，不另写一份光学公式。

## 第一问：给定镜场评价

状态：已实现。

入口：`src/solve_q1.py`

工作内容：

1. 读取 `task/A/fj.xlsx` 的 1745 个坐标；
2. 按给定的 `6.2 m × 6.2 m`、安装高度 `4.5 m` 和中心塔计算 60 个时刻；
3. 输出逐时刻、月平均和年平均结果；
4. 完成反射定律、效率范围、单镜遮挡、集热器尺寸和采样收敛检查。

第一问只做评价，不改变塔位置、镜面尺寸、数量和坐标。

## 第二问：统一规格镜场优化

状态：待实现。

建议入口：`src/solve_q2.py`

设计变量：

- 吸收塔位置 `(x_T, y_T)`；
- 统一镜面宽度、高度和安装高度；
- 定日镜数量；
- 各定日镜位置。

约束：

- 年平均输出热功率不低于 `42 MW`；
- 镜面边长、安装高度、场地边界、塔周禁布区和最小中心距满足题面；
- 所有镜面使用统一尺寸和安装高度。

目标：

```text
最大化 单位镜面面积年平均输出热功率
```

实现顺序：

1. 用参数化环形/分区布局生成合法候选镜场；
2. 用低精度光学评价进行全局搜索；
3. 对候选解调整镜面数量，使功率约束贴边；
4. 用第一问正式精度复算；
5. 写出论文表 1、2、3 和 `task/A/result2.xlsx` 对应的完整结果文件。

## 第三问：可变规格镜场优化

状态：待实现。

建议入口：`src/solve_q3.py`

在第二问变量基础上，每面定日镜允许拥有不同的：

- 镜面宽度和高度；
- 安装高度。

直接优化数千个连续变量不可控，建议先按半径、方位或效率分区，让同一区域共享一组尺寸和高度；得到稳定方案后，再对边界镜和低效率镜做局部调整。

实现顺序：

1. 复用第二问的塔位置和布局生成器；
2. 增加分区规格参数；
3. 以总面积最小/单位面积功率最大为目标做分层优化；
4. 剔除低贡献镜并重新满足 `42 MW`；
5. 用正式精度复算并写出 `result3.xlsx`。
