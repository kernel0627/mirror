# 第三问 Campo2D 技术说明

## 1. 独立边界

新实现位于 `src/heliostat/q3_campo2d/`，入口为
`src/solve_q3_campo2d.py`，输出为 `outputs/q3_campo2d/`。它不读取六组
参数作为初值，也不覆盖 `outputs/q3/` 或 `outputs/q3_continuous/`。

问题二的 1469 面正式坐标仅用于：

- 确认基准圆环前缀为 28 环；
- 自动识别第 28 环被删除的一对结构化镜位；
- 验证统一规格起点逐坐标复现问题二镜场。

## 2. 文件职责

| 文件 | 职责 |
| --- | --- |
| `model.py` | 动态 Campo 重建、结构化排除、五节点基函数、角度特征和异构几何 |
| `evaluate.py` | 粗、中、正式、加密和 smoke 精度下的完整光学评价与缓存 |
| `search.py` | Sobol 初值筛选、分块 best-improvement 和多起点联合回扫 |
| `export.py` | 02–16 数据、JSON、Markdown 和 `result3.xlsx` |
| `plot.py` | 图 3-1 至图 3-4 |
| `solve.py` | 多起点搜索、正式候选选择、加密验证和导出编排 |

## 3. 运行命令

专项测试：

```bash
conda run -n agent env PYTHONPATH=src \
python -m unittest discover -s tests -p 'test_q3_campo2d.py' -v
```

端到端 smoke：

```bash
conda run -n agent env MPLCONFIGDIR=/tmp/codex-matplotlib PYTHONPATH=src \
python src/solve_q3_campo2d.py \
  --smoke \
  --sobol-count 1 \
  --retained-starts 2 \
  --max-rounds 1 \
  --max-joint-cycles 1 \
  --medium-candidates 1 \
  --output /tmp/q3-campo2d-smoke
```

正式运行：

```bash
conda run -n agent env MPLCONFIGDIR=/tmp/codex-matplotlib PYTHONPATH=src \
python src/solve_q3_campo2d.py
```

正式搜索启动前，程序先以 `15×15` 阴影网格、256 条截断光线和完整 60 个
状态回归问题二统一规格基线；镜子数、总面积、年平均功率或单位面积输出超出
容差时立即停止，不进入搜索。

正式运行完成后生成单文件展示稿：

```bash
conda run -n agent python tool/build_q3_campo2d_bundle.py
```

## 4. 精度与结果状态

正式默认使用 16 个 Sobol 起点，保留 3 个起点，每轮粗精度排序后将前 4 个
候选送中精度复算；每个参数步长最多运行 4 轮，联合回扫最多运行 6 轮。

当前 8 项专项测试和 smoke 已通过。smoke 只包含六月正午一个状态；其功率、
面积和 $q$ 不属于正式年平均结果。正式多起点搜索尚未运行。
