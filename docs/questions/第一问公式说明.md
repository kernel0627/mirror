# 第一问公式说明

本文档集中列出第一问从太阳位置、镜面姿态、四项效率到功率汇总的完整公式
体系。建模过程、数值配置和结果见 [`第一问.md`](第一问.md)。

## 1. 坐标、索引与主要符号

采用东—北—天右手坐标系： $x:\text{东},\qquad y:\text{北},\qquad z:\text{竖直向上}.$ 主要索引和符号如下。

| 符号 | 含义 |
| --- | --- |
| $i=1,\ldots,N$ | 定日镜编号，第一问 $N=1745$ |
| $m=1,\ldots,12$ | 月份 |
| $\tau\in\{9,10.5,12,13.5,15\}$ | 当地太阳时 |
| $\boldsymbol c_i$ | 第 $i$ 面镜子中心 |
| $\boldsymbol C$ | 集热器中心 |
| $\boldsymbol s_{m,\tau}$ | 镜面指向太阳的单位向量 |
| $\boldsymbol r_i$ | 镜面指向集热器中心的单位向量 |
| $\boldsymbol n_{i,m,\tau}$ | 镜面法向 |
| $w,h,H$ | 镜面宽、高和镜面中心安装高度 |
| $\eta_{\cos},\eta_{\mathrm{sb}},\eta_{\mathrm{at}},\eta_{\mathrm{trunc}}$ | 四项光学效率 |
| $\eta_{\mathrm{ref}}$ | 镜面反射率 |
| $P_{i,m,\tau}$ | 单镜输出热功率 |
| $P_{m,\tau}$ | 镜场输出热功率 |

第一问固定参数为 $N=1745,\qquad w=h=6.2\ \mathrm{m},\qquad H=4.5\ \mathrm{m},$ $\boldsymbol C=(0,0,86),\qquad R_{\mathrm{rec}}=4\ \mathrm{m},\qquad 82\le z\le90,$ $\eta_{\mathrm{ref}}=0.92,\qquad \varphi=39.4^\circ,\qquad H_{\mathrm{site}}=3\ \mathrm{km}.$

## 3. 日期、太阳赤纬与时角

### 3.1 春分日起算天数

以 3 月 21 日为第 0 天： $\boxed{ D_m=\operatorname{DOY}(m,21)-80 }.$ 其中 $\operatorname{DOY}$ 为一年中的日序号。

### 3.2 太阳赤纬

太阳赤纬为 $\boxed{ \delta_m = \arcsin\!\left[ \sin\!\left(\frac{2\pi D_m}{365}\right) \sin(23.45^\circ) \right] }.$

### 3.3 太阳时角

$\boxed{ \omega_\tau = \frac{\pi}{12}(\tau-12) }.$ 正午 $\tau=12$ 时有 $\omega=0$；上午时角为负，下午为正。

---

## 4. 太阳方向、高度角与方位角

在东—北—天坐标系中，

$$
\boxed{
\boldsymbol s_{m,\tau}
=
\begin{bmatrix}
s_E\\s_N\\s_U
\end{bmatrix}
=
\begin{bmatrix}
-\cos\delta_m\sin\omega_\tau\\
\cos\varphi\sin\delta_m
-\sin\varphi\cos\delta_m\cos\omega_\tau\\
\sin\varphi\sin\delta_m
+\cos\varphi\cos\delta_m\cos\omega_\tau
\end{bmatrix}
}.
$$

数值计算后再执行归一化： $\boldsymbol s\leftarrow \frac{\boldsymbol s}{\|\boldsymbol s\|}.$ 太阳高度角为 $\boxed{ \alpha_{m,\tau}=\arcsin(s_U) },$ 太阳方位角以北向为零、向东为正： $\boxed{ \gamma_{m,\tau} = \operatorname{atan2}(s_E,s_N) \pmod{2\pi} }.$

---

## 5. 法向直接辐射强度 DNI

设场地海拔为 $H_{\mathrm{site}}$，单位为 km，则 $a = 0.4237 -0.00821(6-H_{\mathrm{site}})^2,$ $b = 0.5055 +0.00595(6.5-H_{\mathrm{site}})^2,$ $c = 0.2711 +0.01858(2.5-H_{\mathrm{site}})^2.$ 太阳位于地平线上方时， $\boxed{ DNI_{m,\tau} = 1.366 \left[ a+b\exp\!\left( -\frac{c}{\sin\alpha_{m,\tau}} \right) \right] \quad \mathrm{kW/m^2} }.$ 若 $\alpha_{m,\tau}\le0$，则令 $DNI=0$。

---

## 6. 镜面位置、目标方向与法向

第 $i$ 面定日镜中心为 $\boxed{ \boldsymbol c_i=(x_i,y_i,H) }.$ 镜面中心到集热器中心的距离为 $\boxed{ d_i=\|\boldsymbol C-\boldsymbol c_i\| }.$ 镜面指向集热器中心的单位向量为 $\boxed{ \boldsymbol r_i = \frac{\boldsymbol C-\boldsymbol c_i}{d_i} }.$ 镜面法向取太阳方向与目标方向的单位角平分线： $\boxed{ \boldsymbol n_{i,m,\tau} = \frac{ \boldsymbol s_{m,\tau}+\boldsymbol r_i }{ \|\boldsymbol s_{m,\tau}+\boldsymbol r_i\| } }.$ 若入射方向为 $\boldsymbol d_{\mathrm{in}}=-\boldsymbol s$，则反射公式为 $\boxed{ \boldsymbol d_{\mathrm{refl}} = \boldsymbol d_{\mathrm{in}} -2(\boldsymbol d_{\mathrm{in}}\cdot\boldsymbol n)\boldsymbol n }.$ 中心光线应满足 $\boldsymbol d_{\mathrm{refl}}=\boldsymbol r_i.$

---

## 7. 镜面局部坐标系

令竖直方向 $\boldsymbol k=(0,0,1),$ 镜面宽度轴为 $\boxed{ \boldsymbol e_{w,i} = \frac{\boldsymbol k\times\boldsymbol n_i} {\|\boldsymbol k\times\boldsymbol n_i\|} },$ 高度轴为 $\boxed{ \boldsymbol e_{h,i} = \frac{\boldsymbol n_i\times\boldsymbol e_{w,i}} {\|\boldsymbol n_i\times\boldsymbol e_{w,i}\|} }.$ 镜面局部坐标 $(u,v)$ 对应的三维点为 $\boxed{ \boldsymbol p_i(u,v) = \boldsymbol c_i +u\boldsymbol e_{w,i} +v\boldsymbol e_{h,i} },$ 其中 $-\frac w2\le u\le\frac w2, \qquad -\frac h2\le v\le\frac h2.$

---

## 8. 余弦效率

$\boxed{ \eta_{\cos,i,m,\tau} = \max\!\left( 0, \boldsymbol n_{i,m,\tau}\cdot \boldsymbol s_{m,\tau} \right) }.$ 由于法向是太阳方向和目标方向的角平分方向，该式也等于入射光线与镜面法向夹角余弦的绝对值。

---

## 9. 阴影遮挡效率

### 9.1 镜面规则网格

正式计算取 $G=15.$ 第 $a,b$ 个小格中心的局部坐标为 $u_a = -\frac w2 +\left(a+\frac12\right)\frac wG, \qquad a=0,\ldots,G-1,$ $v_b = -\frac h2 +\left(b+\frac12\right)\frac hG, \qquad b=0,\ldots,G-1.$ 采样点为 $\boldsymbol p_{i,a,b} = \boldsymbol p_i(u_a,v_b).$

### 9.2 射线与有限矩形镜面求交

从采样点 $\boldsymbol p$ 沿单位方向 $\boldsymbol d$ 发射射线 $\boldsymbol x(\lambda) = \boldsymbol p+\lambda\boldsymbol d, \qquad \lambda>0.$ 与第 $j$ 面镜子所在平面的交点参数为 $\boxed{ \lambda_j = \frac{ (\boldsymbol c_j-\boldsymbol p)\cdot\boldsymbol n_j }{ \boldsymbol d\cdot\boldsymbol n_j } }.$ 交点为 $\boldsymbol q_j = \boldsymbol p+\lambda_j\boldsymbol d.$ 令 $\Delta\boldsymbol q_j = \boldsymbol q_j-\boldsymbol c_j,$ 若 $\lambda_j>\varepsilon,$ $\left| \Delta\boldsymbol q_j\cdot\boldsymbol e_{w,j} \right| \le\frac w2+\varepsilon,$ $\left| \Delta\boldsymbol q_j\cdot\boldsymbol e_{h,j} \right| \le\frac h2+\varepsilon,$ 则射线命中第 $j$ 面有限矩形镜面。

### 9.3 入射阴影与反射遮挡

正式计算先把候选邻镜限制为 $\boxed{ \mathcal N_i = \left\{ j\ne i: \left\| (x_j,y_j)-(x_i,y_i) \right\|_2 \le60\ \mathrm{m} \right\} }.$ 对第 $i$ 面镜子的采样点：

- 沿 $\boldsymbol s_{m,\tau}$ 检测入射阴影；
- 沿 $\boldsymbol r_i$ 检测到集热器之前的反射遮挡。

因此，反射遮挡射线除了满足矩形求交条件外，还要求 $\boxed{ 0<\lambda_j<d_i }.$ 定义二值变量 $B^{\mathrm{in}}_{i,\ell} \in\{0,1\}, \qquad B^{\mathrm{out}}_{i,\ell} \in\{0,1\}.$ 两类损失取并集： $B_{i,\ell} = \max\!\left( B^{\mathrm{in}}_{i,\ell}, B^{\mathrm{out}}_{i,\ell} \right).$ 阴影遮挡效率为 $\boxed{ \eta_{\mathrm{sb},i,m,\tau} = 1-\frac{1}{G^2} \sum_{\ell=1}^{G^2}B_{i,\ell} }.$

---

## 10. 大气透射率

$\boxed{ \eta_{\mathrm{at},i} = \operatorname{clip}_{[0,1]} \left( 0.99321 -0.0001176d_i +1.97\times10^{-8}d_i^2 \right) }.$ 第一问镜位固定，因此该项不随月份和时刻变化。

---

## 11. 截断效率

### 11.1 Sobol 联合样本

正式计算每镜每时刻取 $L=256$ 个固定四维 Sobol 样本 $\boldsymbol u_\ell = (u_{\ell,1},u_{\ell,2},u_{\ell,3},u_{\ell,4}) \in[0,1)^4.$ 镜面采样坐标为 $u_\ell^{(m)} = \left(u_{\ell,1}-\frac12\right)w,$ $v_\ell^{(m)} = \left(u_{\ell,2}-\frac12\right)h.$ 镜面采样点为 $\boldsymbol o_{i,\ell} = \boldsymbol c_i +u_\ell^{(m)}\boldsymbol e_{w,i} +v_\ell^{(m)}\boldsymbol e_{h,i}.$

### 11.2 太阳圆盘方向

设太阳角半径为 $\rho_\odot=4.65\times10^{-3}\ \mathrm{rad}.$ 在垂直于太阳中心方向的平面中构造正交基 $\boldsymbol t_1,\qquad\boldsymbol t_2.$ 令 $\theta_\ell = \rho_\odot\sqrt{u_{\ell,3}}, \qquad \psi_\ell = 2\pi u_{\ell,4},$ $\boldsymbol t_\ell = \cos\psi_\ell\,\boldsymbol t_1 +\sin\psi_\ell\,\boldsymbol t_2.$ 太阳圆盘样本方向为 $\boxed{ \boldsymbol s_\ell = \cos\theta_\ell\,\boldsymbol s +\sin\theta_\ell\,\boldsymbol t_\ell }.$

### 11.3 样本反射方向

样本入射方向为 $\boldsymbol d_{\mathrm{in},\ell} = -\boldsymbol s_\ell$，反射方向为

$$
\boxed{
\boldsymbol d_{\mathrm{refl},i,\ell}
=
\boldsymbol d_{\mathrm{in},\ell}
-2\left(
\boldsymbol d_{\mathrm{in},\ell}
\cdot\boldsymbol n_i
\right)\boldsymbol n_i
}.
$$

### 11.4 射线与有限圆柱集热器求交

设射线 $\boldsymbol x(\lambda) = \boldsymbol o+\lambda\boldsymbol d, \qquad \lambda>0.$ 圆柱轴线通过 $(x_T,y_T)$，半径为 $R_{\mathrm{rec}}$。令 $x_0=o_x-x_T,\qquad y_0=o_y-y_T,$ 则圆柱侧面求交方程为 $A\lambda^2+B\lambda+C=0,$ 其中 $A=d_x^2+d_y^2,$ $B=2(x_0d_x+y_0d_y),$ $C=x_0^2+y_0^2-R_{\mathrm{rec}}^2.$ 判别式为 $\Delta=B^2-4AC.$ 若存在正根 $\lambda_\pm = \frac{-B\pm\sqrt{\Delta}}{2A}$ 使得 $z_{\min} \le o_z+\lambda_\pm d_z \le z_{\max},$ 则该样本命中集热器，记 $I^{\mathrm{rec}}_{i,\ell}=1;$ 否则记为 0。

截断效率为 $\boxed{ \eta_{\mathrm{trunc},i,m,\tau} = \frac{1}{L} \sum_{\ell=1}^{L} I^{\mathrm{rec}}_{i,\ell} }.$

---

## 12. 综合光学效率

第 $i$ 面镜子在状态 $(m,\tau)$ 下的综合光学效率为

$$
\boxed{
\eta_{i,m,\tau}
=
\eta_{\cos,i,m,\tau}
\cdot\eta_{\mathrm{sb},i,m,\tau}
\cdot\eta_{\mathrm{at},i}
\cdot\eta_{\mathrm{trunc},i,m,\tau}
\cdot
\eta_{\mathrm{ref}}
}.
$$

各项效率取值都必须位于 $0\le\eta\le1.$

---

## 13. 单镜与镜场输出热功率

单镜面积为 $A_m=wh.$ 单镜输出热功率为 $\boxed{ P_{i,m,\tau} = DNI_{m,\tau} \cdot A_m \cdot \eta_{i,m,\tau} }.$ 由于 $DNI$ 的单位为 $\mathrm{kW/m^2}$，所以
$P_{i,m,\tau}$ 和 $P_{m,\tau}$ 的计算单位均为 $\mathrm{kW}$。

镜场输出热功率为 $\boxed{ P_{m,\tau} = \sum_{i=1}^{N} P_{i,m,\tau} }.$ 因为第一问所有镜面面积相同，状态平均效率为 $\boxed{ \overline\eta_{m,\tau} = \frac1N \sum_{i=1}^{N} \eta_{i,m,\tau} }.$ 单位镜面面积输出热功率为 $\boxed{ q_{m,\tau} = \frac{P_{m,\tau}}{Nwh} }.$

---

## 14. 月平均、年平均与单镜年平均

对任一状态指标 $X_{m,\tau}$，月平均为 $\boxed{ \overline X_m = \frac15 \sum_{\tau\in\{9,10.5,12,13.5,15\}} X_{m,\tau} }.$ 年平均为 $\boxed{ \overline X_{\mathrm{year}} = \frac1{60} \sum_{m=1}^{12} \sum_{\tau} X_{m,\tau} }.$ 对镜场功率应另外写明输出单位换算： $\boxed{ \overline P_{\mathrm{MW}} = \frac1{1000} \left[ \frac1{60} \sum_{m=1}^{12}\sum_{\tau}P_{m,\tau} \right] }.$ 单位镜面面积年平均输出保持为 $\boxed{ \overline q = \frac{ \frac1{60}\sum_{m=1}^{12}\sum_{\tau}P_{m,\tau} }{Nwh} } \qquad \left(\mathrm{kW/m^2}\right).$ 第 $i$ 面镜子的年平均指标为 $\boxed{ \overline X_i = \frac1{60} \sum_{m=1}^{12} \sum_{\tau} X_{i,m,\tau} }.$

---

## 15. 正式输出

第一问公式系统最终输出：

$$
\boxed{
\overline\eta_{\cos},
\overline\eta_{\mathrm{sb}},
\overline\eta_{\mathrm{at}},
\overline\eta_{\mathrm{trunc}},
\overline\eta,
\overline P_{\mathrm{MW}},
\overline q
}.
$$
