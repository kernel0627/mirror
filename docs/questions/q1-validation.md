# 第一问当前结果与验证

- 定日镜数量：1745；
- 阴影网格：`15×15`；
- 截断光线：每面镜、每时刻 256 条固定 Sobol 联合样本；
- 邻镜搜索半径：`60 m`；
- 计算时刻：12 个月 × 每月 5 个时刻。

## 自动检查

当前 5 项检查均已通过：

1. 3 月 21 日对应赤纬公式的 `D=0`；
2. 上午和下午太阳方向满足对称性；
3. 中心光线反射后指向集热器，误差低于 `10^-12`；
4. 单镜场的阴影遮挡效率为 1；
5. 增大集热器半径不会降低固定样本下的截断效率。
