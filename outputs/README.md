# 输出目录

正式运行第一问后，结果写入 `outputs/q1/`：

- `01_第一问完整代码.py`：把共享光学核心和第一问流程合并后的单文件展示稿；
- `02_逐时刻计算结果.csv`：12 个月 × 5 个时刻的全场结果；
- `03_月平均计算结果.csv`：12 个月的月平均结果；
- `04_年平均计算结果.json`：年平均结果；
- `05_单镜年平均结果.csv`：1745 面镜子的年平均分项效率、综合效率和输出功率；
- `06_运行配置.json`：输入路径、题目参数和数值精度参数；
- `07_论文结果与验证表.md`：月平均表、年平均表和一张数值验证表；
- `08_月平均光学性能与输出热功率.png`：月平均综合效率和单位面积热功率组合图；
- `09_单镜年平均综合光学效率空间分布.png`：实际镜场坐标上的单镜年平均效率分布。

这里不保存全部单镜逐时刻数据；空间图直接读取单镜年平均结果，不重新计算另一套口径。

本目录只保存可直接交付的生成结果和单文件代码展示稿，不放题面、附件、工程源码目录或手工笔记。

第二问正式结果同样直接写入扁平目录 `outputs/q2/`，只保留可交付内容：

- `01_第二问完整代码.py`：共享光学核心、两布局、搜索、验证、输出和绘图合并后的可独立运行单文件；

- `04_月平均计算结果.csv`：最终镜场 12 个月平均结果；
- `05_年平均计算结果.json`：最终镜场年平均结果；
- `06_单镜年平均结果.csv`：最终镜场单镜年平均效率和功率；
- `07_最终方案摘要.json`：布局类型、参数、面积、功率和约束余量；

- `10_第二问提交结果.xlsx`：按题目模板填写的第二问提交文件；
- `11_图2-1_两种候选布局平面分布与单镜年平均输出.png`；
- `12_图2-2_两种候选布局主要性能指标对比.png`；
