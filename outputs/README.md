# 输出文件说明

`outputs/` 保存三问完整计算生成的结果、图表和提交文件。快速检查与调试结果应
写入临时目录，避免与这里的正式数据混用。

## 第一问

`outputs/q1/` 记录给定 1745 面镜场的评价结果：

- `01_第一问完整代码.py`：可独立运行的第一问单文件程序；
- `02_逐时刻计算结果.csv`：12 个月、每月 5 个规定时刻的镜场汇总；
- `03_月平均计算结果.csv`：12 个月的月平均指标；
- `04_年平均计算结果.json`：年平均效率、功率和单位面积输出；
- `05_单镜年平均结果.csv`：1745 面镜子的年平均指标；
- `06_运行配置.json`：物理参数、采样精度和输入路径；
- `07_论文结果与验证表.md`：月平均、年平均和收敛验证表；
- `08_月平均光学性能与输出热功率.png`：月平均结果图；
- `09_单镜年平均综合光学效率空间分布.png`：单镜空间分布图。

## 第二问

`outputs/q2/` 记录两种统一规格布局的比较和最终 Campo 方案：

- `01_第二问完整代码.py`：可独立运行的第二问单文件程序；
- `02_双布局比较.json`：分区交错同心圆与改进 Campo 的正式指标；
- `03_最终镜位坐标.csv`：最终 1469 面镜子的坐标；
- `04_月平均计算结果.csv`：最终方案的月平均指标；
- `05_年平均计算结果.json`：最终方案的年平均指标；
- `06_单镜年平均结果.csv`：最终方案的单镜年平均指标；
- `07_最终方案摘要.json`：布局参数、面积、镜子数和约束结果；
- `08_论文结果与验证表.md`：方案比较、最终参数、月平均和验证表；
- `09_高精度加密验证.json`：更密网格、更多光线下的复算结果；
- `11_*.png` 至 `14_*.png`：布局、指标、月平均和三维结构图；
- `15_双布局月平均对比数据.csv`：月平均对比图的数据；
- `result2.xlsx`：按题目模板生成的第二问提交文件。

## 第三问

`outputs/q3/` 记录六区异构镜场的回归、搜索、收口、验收和边界检验：

- `01_第三问完整代码.py`：可独立运行的第三问单文件程序；
- `02_六组回归结果.json`：原六区方案的回归结果；
- `03_塔位两种语义扫描.csv`：塔位模式 A/B 的扫描结果；
- `04_Campo几何粗扫.csv`：$D_1$、$g$ 及局部组合扫描；
- `05_规格参数敏感性.csv`：六区宽、高和安装高度的正负扰动；
- `06_活跃变量集合.json`：正式复算后保留的搜索变量和预算；
- `07_局部搜索轨迹.csv`：分块变步长搜索的接受与拒绝记录；
- `08_正式候选比较.csv`：入围候选的统一正式精度比较；
- `09_最终六区参数.csv`：最终塔位、Campo 参数和六区规格；
- `10_最终逐镜参数与坐标.csv`：1471 面镜子的坐标和规格；
- `11_正式结果比较.json`：原六区、收口前后及最终方案的正式结果；
- `12_加密验收比较.json`：80 m 和 100 m 邻镜半径下的加密结果；
- `13_几何约束验证.json`：场地、禁区、镜间距和触地约束；
- `14_局部收口检查.csv`：塔位包围、细扫和最细邻域检查；
- `15_论文结果与验证表.md`：正式结果、加密结果、六区规格和边界检验表；
- `16_参数敏感性图.png`：规格变量敏感性；
- `17_六区宽高与安装高度图.png`：六区规格；
- `18_六组与优化方案指标比较图.png`：原六区与最终方案的指标比较；
- `19_最终六区镜场与塔位平面图.png`：最终镜场、六区和塔位；
- `20_六区边界局部敏感性检验.csv`：18 个边界候选的中精度与正式精度结果；
- `21_六区边界局部敏感性图.png`：边界移动后的功率余量、单位面积输出变化和分类；
- `result3.xlsx`：按题目模板生成的第三问提交文件。

第三问最终方案含 1471 面镜子，正式年平均输出热功率为
$42.086214740\ \mathrm{MW}$，单位镜面面积年平均输出为
$0.693125957\ \mathrm{kW/m^2}$。80 m 和 100 m 加密结果均为
$42.076085938\ \mathrm{MW}$ 和 $0.692959144\ \mathrm{kW/m^2}$。
